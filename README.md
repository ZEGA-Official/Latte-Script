# Latte-Script

## Grammar & Syntax

Latte-Script uses an easy grammar so to help when trying to learn, here are some examples:

```swift
class Main
    place = "World"
    func "HelloWorld"() {
        print "Hello" + place
    }
```

```python
imp math

1 * 2 = a
if a > 1 then:
    print "A is more than 1"
elif a == 1 then:
    print "A is 1"
else:
    print "A is less than 1"
```
